## LSP_ssu_crontab-rsync  
### 2020년 컴퓨터학부 리눅스시스템프로그래밍 설계과제 3

    - [ssu_crontab.c 사용방법]
      1. git clone https://www.github.com/yegyeom/LSP_ssu_crontab-rsync.git
      2. cd LSP_ssu_crontab-rsync
      3. make
      4. ./ssu_crontab
      5. 프롬프트에서 주기적으로 실행할 명령어 입력  
    - [ssu_rsync.c 사용방법]
      1. git clone https://www.github.com/yegyeom/LSP_ssu_crontab-rsync.git
      2. cd LSP_ssu_crontab-rsync
      3. make
      4. ./ssu_rsync <src> <dir>
